# Classification des Fleurs d'Iris
Ce projet a pour objectif de construire un modèle simple d’apprentissage automatique qui permet de classifier les fleurs d’Iris en trois espèces : Setosa, Versicolor et Virginica, à partir de leurs caractéristiques (longueur et largeur des sépales et pétales).
Ce travail s’inscrit dans le cadre du TP "Développement collaboratif et open science".
Il permet d’utiliser Git et GitHub pour le versionnement, ainsi que Jupyter Notebook avec NumPy, Pandas et Scikit-learn pour documenter et réaliser une analyse reproductible.
